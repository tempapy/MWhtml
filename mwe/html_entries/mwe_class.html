CLASS [098-b], s. (A rank or order of persons) varṇaḥ, vargaḥ, padaṃ, āśramaḥ, jātiḥ f. — (A set of things or beings) gaṇaḥ, paṃktiḥ f., śreṇī, saṃsargaḥ, samūhaḥ, gotraṃ; ‘class of verbs,’ gaṇaḥ. — (A number of scholars) sahādhyāyināṃ paṃktiḥ, chātrasamūhaḥ; ‘the lower classes,’ avaravarṇāḥ m. pl., antyajātīyāḥ m. pl.
