• abhidru 1 P.
      1 (a) To run up to, run near; payasyabhidravati bhuvaṃ yugāvadhau Śi. 17. 40. (b) To invade, march against, fall upon, attack, assail; makaraṃdena pratihato jāmātā balātkāreṇa abhidravan Māl. 7 laying violent hands on; gajā ivānyonyamabhidravaṃtaḥ (vāridharāḥ) Mk. 5. 21; Ve. 3.
      2 To overrun,; infest, harass, afflict; janmamṛtyujarāvyādhivedanābhiramidrutaṃ (dehaṃ) Mb.
      3 To come over, pass or run over.
      4 To befall. --Caus. To rout, put to flight; purarakṣānabhidrāvya Dk. 14. [0130-a]
