• deya a. [dā karmaṇi yat] 1 To be given, offered or presented; R. 3. 16. --2 Fit to be given, proper for a gift. --3 To be returned or restored; vibhāvitaikadeśena deyaṃ yadabhiyujyate V. 4. 17; Ms. 8. 139, 185. --4 To be shown. --5 To be given in marriage. --6 To be paid (as a debt &c.). --7 To be placed, put, applied, laid, &c. see dā above. --yaṃ A gift, donation.
