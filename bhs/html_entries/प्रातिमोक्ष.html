• prātimokṣa, m. (on etym. and literal mg. cf. Winter- nitz, H.I.Lit. 2.22; = Pali pātimokkha, pāṭi°), n. of the code of precepts in the Vinaya according to which monks are controlled and corrected (with penances etc.): Mvy 9217 = Tib. so sor thar pa, individual (as if = prati) release (sc. from sin), which is accomplished by confession and penance. As in Pali, cpd. with saṃvara, q.v. for this cpd. Generally seems to refer to the literary text containing this code, one version of which is our Prāt: °kṣa-sūtrodde- śam (acc.) Prāt 476.8; Av ii.21.12; °śaḥ MSV iii.107.9; prātimokṣād Śikṣ 125.9 (seems to refer to a literary text); pūrāgheṇa (see s.v. pūra) prātimokṣe Śikṣ 143.6--7, in the (book of) expiation according to full value (of the require- ments, i.e. matching the offense). See also pratimokṣa. [391,2]
