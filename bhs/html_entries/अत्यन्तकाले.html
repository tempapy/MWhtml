• atyantakāle, adv. (cf. AMg. accantakālassa, Ratnach.), for a very long time: Mv i.103.9, read yadā (mss., Senart omits this) preteṣu, atyantakāle (mss. reported °kāye, Senart em. °kāyeṣu; perh. read °kālena?) nopapadyanti, if (they are born) among pretas, they are not born (so) for a very long time.
