• saṃmodanīya, adj. (= Pali id.), = prec.: °yaḥ Mvy 2942 (after saṃrañjanīyaḥ 2941); f. °yā, with kathā, Mv repeatedly, see s.v. saṃmodate, °dayati. In Karmav 27.1 text substitutes sukha-saṃbhāvanāyāṃ, which might be em. to °bhāvanāṃ, causing pleasure; but this is otherwise unrecorded in this cliché, and Lévi suggests in a note saṃmodanīyāṃ.
