• śodhya mfn. (-dhyaḥ-dhyā-dhyaṃ)
    1 To be purified, cleaned, refined, &c.
    2 Payable, due.
    3 To be subtracted. m. (-dhyaḥ) An accused person, one to be justified or tried. n. (-dhyaṃ) A constant number to be subtracted in some astronomical computations.
    E. śudha to be pure, yata aff.
